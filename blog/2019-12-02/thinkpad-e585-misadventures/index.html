Thinkpad E585 (Mis)adventures

Posted on 2019-12-02

Notes on trying to get the Lenovo Thinkpad E585 to work on linux.

Had to update the BIOS to get Linux to boot at all, and also updated the NVME driver using info from here.

  • NVME SSD Model: LENSE30256GMSP34MEAT3TA
  • Current BIOS version: R0UET76W (1.56)
  • Current NVME Firmware: 1.6.0412
  • Swap: swapfile used
  • BIOS settings: made no difference

Arch Linux

  • Filesystem: btrfs on LUKS
  • DM/DE: xorg, LightDM, Cinnamon
  • Boot params: irqpoll irqfixup
  • OS updated on: 2019-12-01

Kernels Tested:

  • linux-5.3.13-arch-1
  • linux-5.4.1-arch-1 (trunk pkgbuild)
  • linux-amd-raven 5.3.v.14-1 (AUR)

Suspend

Seems to work fine

Hibernate (uswsusp)

Does not turn off after hibernating, only works intermittently, sometimes crashing before hibernate. Additionally TTYs stay blank after hibernate.

Hibernate (swsusp)

Boot params: irqpoll irqfixup resume=[...] resume_offset=[...]

Doesn't work at all (Haven't put too much effort into it)

Linux Mint 19.2

  • Kernel: 4.15.0-54-generic
  • Filesystem: ext4
  • Boot params: irqpoll irqfixup resume=[...] resume_offset=[...]

Suspend

Seems to work fine

Hibernate (swsusp)

Doesn't work, black screen after restart
